## Webpack4-boilerplate

This project was designed to demonstrate the basic usage of Webpack4. Topics includes:

- Production environment setup
  - branch mode
- Loaders
  - branch loader
- Plugins
   - branch plugin
- Multipage application config
  - branch multi-page
- Single page application config
  - branch single-page
